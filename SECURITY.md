# Security Policy

## Reporting a Vulnerability

If you discover a security vulnerability in TruthLayer, please email security@truthlayer.io.

Do not create a public GitHub issue for security vulnerabilities.

## Scope

- Chrome Extension (all files in `extension/`)
- Backend API (all files in `backend/`)
- Web Dashboard (all files in `web/`)

## Supported Versions

| Version | Supported |
|---------|-----------|
| 1.x     | ✅ |
